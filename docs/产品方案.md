# ScentSoul 产品方案

## 一句话定位

ScentSoul 是一套面向香水品牌的数字 IP 与用户运营系统。用户购买香水后，扫描瓶身二维码/NFC 解锁专属数字香气灵魂，获得与香调匹配的动态视觉、音乐、故事和收藏卡。

## 用户痛点

### 消费者

- 买香水的情绪价值强，但购买后的互动很短。
- 香水很难被分享，照片和文字不能充分表达气味。
- 用户希望自己的选择有专属感，而不是所有人都拿到同一张品牌物料。

### 品牌

- 线下试香和线上种草割裂，购买后难以继续触达用户。
- 香水用户偏好、情绪标签、复购周期难沉淀。
- 新品营销常停留在海报和短视频，缺少可收藏、可分享、可二次唤醒的互动资产。

## 核心用户路径

1. 用户购买香水。
2. 扫描瓶身二维码或 NFC。
3. 输入昵称，选择今天的气味情绪。
4. 大模型生成专属数字 IP。
5. 前端渲染动态视觉和音乐。
6. 用户收藏或分享香气卡片。
7. 后续通过节日、天气、使用次数解锁新动画和新音乐。

## MVP 功能

### 用户端

- 香水选择
- 瓶身编号输入/随机生成
- 情绪选择
- 数字 IP 生成
- 动态场景展示
- 香气音乐播放
- 收藏卡复制/保存
- 分享话题生成

### 品牌运营端

- 扫码激活量
- 分享率
- 二次唤醒率
- 用户偏好情绪
- 香型偏好分布
- 可运营节点：生日、节日、天气、复购周期

## 香调到视听的映射

| 香调 | 视觉 | 动画 | 音乐 |
| --- | --- | --- | --- |
| 柑橘水生绿意 | 水流、青草、柑橘弧线、莲花 | 流动、轻快、透明 | 中高 BPM、玻璃音、清亮合成器 |
| 花香粉感麝香 | 花瓣、薄雾、丝带、柔光 | 慢速绽放、漂浮 | 慢 BPM、柔和 Pad、弦乐质感 |
| 木质琥珀皮革 | 木纹、纸页、火光、烟雾 | 低速漂移、余烬闪烁 | 低 BPM、低频、温暖 Pad |
| 茶香绿叶雨水 | 雨线、竹叶、蒸汽、水纹 | 下落、回环、呼吸 | 中慢 BPM、雨声、空气感音色 |

## 个性化生成机制

同一款香水保持同一世界观，但每瓶有差异：

- 瓶身编号影响 IP 名称、稀有度、颜色细节。
- 用户昵称影响故事称呼和收藏卡文案。
- 情绪选择影响视觉运动速度、音乐亮度和故事方向。
- 香调决定主视觉元素和音乐基调。

## 后续迭代

1. 接入真实二维码/NFC。
2. 增加用户登录和收藏库。
3. 增加隐藏卡面、节日活动、天气联动。
4. 增加品牌后台，支持活动配置和用户分层。
5. 支持 AI 生成专属短视频和可下载动态壁纸。
