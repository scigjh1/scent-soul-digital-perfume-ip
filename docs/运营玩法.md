# 产品运营玩法

## 运营目标

ScentSoul 不只是一个扫码动效，而是品牌购买后的用户运营入口。

核心目标：

- 提升购买仪式感
- 提升用户分享率
- 沉淀香型偏好和情绪标签
- 增加二次唤醒和复购机会
- 帮助新品发售做话题传播

## 可运营指标

- 扫码激活量
- 激活率：购买后扫码用户 / 购买用户
- 分享率：分享收藏卡用户 / 激活用户
- 二次唤醒率：7 天内再次进入用户 / 激活用户
- 香型偏好：柑橘、水生、花香、木质、茶香等
- 情绪偏好：清透、明亮、潮湿、温柔、神秘、顽皮
- 复购转化：被活动触达后再次购买用户

## 冷启动玩法

### 1. 新品限量编号

新品首批 1000 瓶开放隐藏稀有度：

- Common
- Uncommon
- Rare
- Secret

用户扫码后生成自己的数字 IP，天然适合分享。

### 2. 小红书分享卡

生成收藏卡文案：

```text
我抽到的是尼罗河花园的「河光青芒」
它是清透型气味灵魂，编号 SS-203915
像青芒果、青草和水流一起醒来
```

### 3. 天气联动

雨天唤醒水生调，解锁雨声变奏。

晴天唤醒柑橘调，解锁高亮卡面。

夜晚唤醒木质调，解锁低频版本。

### 4. 复购提醒

香水使用 30 天后，推送：

```text
你的数字灵魂记录了 7 次清透情绪，要不要试试同系列的雨后茶室？
```

## 商业化方向

- 品牌联名数字卡
- 限量隐藏动画
- 会员等级和收藏图鉴
- 香水小样兑换
- 节日限定音乐和视觉
- 线下快闪扫码互动

## 面试表达

```text
这个项目不是简单做一个扫码动画，而是把香水购买后的情绪价值产品化。
用户端获得专属感和分享素材，品牌端获得会员触点、用户偏好和复购运营空间。
```
