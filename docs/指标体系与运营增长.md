# 指标体系与运营增长

## 北极星指标

```text
香水数字 IP 有效激活数
```

有效激活不是简单扫码，而是完成：

```text
扫码 -> 生成数字 IP -> 停留体验 -> 收藏/分享/再次唤醒其中至少一个动作
```

## 漏斗指标

| 阶段 | 指标 | 说明 |
| --- | --- | --- |
| 购买后触达 | 扫码率 | 扫码用户 / 购买用户 |
| 激活 | 生成完成率 | 成功生成 IP 用户 / 扫码用户 |
| 体验 | 平均停留时长 | 用户在动态页面停留多久 |
| 分享 | 收藏卡保存率 | 保存或复制收藏卡用户 / 生成用户 |
| 传播 | 分享率 | 分享到社交平台用户 / 生成用户 |
| 留存 | 7 日二次唤醒率 | 7 天内再次进入用户 / 生成用户 |
| 转化 | 复购意向率 | 点击推荐、小样或礼盒用户 / 二次唤醒用户 |

## 用户分层

### 情绪分层

- 清透：偏好水生、茶香、夏季清爽内容。
- 明亮：偏好柑橘、果香、新品尝鲜内容。
- 温柔：偏好花香、麝香、约会场景。
- 神秘：偏好木质、琥珀、夜间香氛。
- 顽皮：偏好甜香、联名、限定卡面。

### 行为分层

- 激活未分享：推送更好看的收藏卡模板。
- 分享未复访：用评论区互动或隐藏卡面召回。
- 多次唤醒：推荐会员、同系列小样或礼盒。
- 高分享用户：邀请参与新品内测或线下快闪。

## 运营活动设计

### 新品限量编号

首批香水开放隐藏稀有度，用户扫码后抽到不同数字 IP。

目标：

- 提升扫码激活率
- 制造社交分享素材
- 为新品建立第一波 UGC

### 晒出我的香气灵魂

用户分享收藏卡到小红书/朋友圈，带话题参与活动。

目标：

- 提升分享率
- 增加品牌搜索和自然流量
- 收集用户自发表达

### 7 日唤醒隐藏卡面

连续 7 天进入页面，解锁第二段动画、音乐或隐藏卡面。

目标：

- 提升二次唤醒率
- 培养会员互动习惯
- 为复购推荐创造触点

### 天气联动变奏

雨天、水生调、茶香调触发特殊动画；夜间木质调触发低频版本。

目标：

- 增加用户再次打开理由
- 让数字 IP 和真实生活场景绑定

## 复购策略

根据用户香型和情绪偏好推荐：

- 同系列香水
- 旅行装
- 小样组合
- 节日礼盒
- 线下试香活动

示例话术：

```text
你的数字灵魂已经记录了 5 次“清透”情绪。
也许你会喜欢同系列的「雨后茶室」：更安静、更绿、更像雨后的空气。
```

## 面试表达

```text
我把这个项目的运营指标设计成一个完整漏斗：
从购买后扫码、数字 IP 生成、收藏卡分享，到二次唤醒和复购推荐。
这样产品功能不是孤立体验，而是每一步都能产生可运营的数据和动作。
```
