# 功能清单与交付物

## 已完成 Demo 功能

### 用户端功能

- 香水选择：尼罗河花园、绯雾花庭、琥珀书房、雨后茶室。
- 瓶身编号：支持输入编号和随机生成编号。
- 情绪选择：清透、明亮、潮湿、温柔、神秘、顽皮。
- 数字 IP 生成：生成 IP 名字、人格、口号、故事。
- 动态视觉：Canvas 根据香调和模型参数实时渲染。
- 香气音乐：Web Audio 根据 BPM、调式、亮度、温暖度生成氛围音。
- 收藏卡：生成标题、副标题、话题和标签。
- 分享动作：支持复制收藏卡文案。
- 运营方案：支持复制当前用户的运营方案。

### 产品运营功能

- 运营看板：扫码激活、分享率、二次唤醒、偏好情绪。
- 用户分层：根据香型、情绪和互动方式生成用户分层。
- 活动主题：生成新品限量、晒卡、7 日唤醒、天气联动等活动。
- 转化目标：为每次生成结果匹配运营目标。
- 下一步动作：给出复购推荐、小样推荐或会员活动建议。

### 技术功能

- Node.js 本地服务。
- DeepSeek/通义千问/Kimi/智谱等国内大模型兼容接入入口。
- 本地生成器兜底。
- 前端无 API Key 暴露。
- GitHub 开源结构。
